Nota a pie de página 3.1.2.

Todos los cuerpos finitos tienen un número de elementos \(q = p^n\text{,}\) para algún número primo \(p\) y algún entero positivo \(n\text{.}\) Para cada cardinalidad \(q\) así definida hay una y solo una manera posible de definir un campo finito, por lo que todos los campos finitos del mismo orden son isomorfos entre sí [wikipedia].
